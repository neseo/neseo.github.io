proxychains

首先安装proxychains：sudo apt install -y proxychains。

然后编辑 /etc/proxychains.conf 文件，在最后的[ProxyList]一节中增加代理设置，例如：

socks5 127.0.0.1 1080

接下来，所有希望走代理的命令，前面增加proxychains即可，例如：
proxychains lynx https://youtube.com

Debian switch desktop

debian10 切换默认桌面命令

debian10 (安装多桌面)切换默认桌面命令:
update-alternatives –config x-session-manager
选择 要切换的桌面序号 重启即可

Debian & Win10 grub boot

Linux Debian11如何修改Grub2的开机启动顺序

Windows和Linux双系统，如何修改启动顺序。

1、进入/etc/grub.d目录，将该目录下的“30_os-prober”文件改名为“##_os-prober”。##的取值为06～09之间的数字。

2、执行sudo update-grub。

注意是加sudo否则找不到该命令.
